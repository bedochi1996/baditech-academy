# BadiTech Academy — Arabic Cybersecurity & SOC Training Platform

BadiTech Academy is an Arabic-first cybersecurity learning platform designed to help learners progress from foundations to SOC and Blue Team skills through structured content, labs, simulations, quizzes, and practical exercises.

> **Status:** Portfolio Project
> **Purpose:** Demonstrate cybersecurity education design, SOC lab development, platform architecture, and hands-on Blue Team simulation.

---

## Platform Scope

- Certification-oriented learning tracks
- SOC career path: Foundations → SOC L1 → SOC L2 → SOC L3
- Practical cybersecurity labs
- Linux terminal simulator
- Splunk SIEM simulator
- Alert Triage simulator
- CTF challenges
- Progress tracking
- Certificates
- AI learning assistant
- Arabic RTL interface

---

## Blue Team Simulators

- Splunk SIEM Simulator
- Alert Triage Simulator
- SOC Labs
